# idea常用快捷键

## 公共
### 搜索与替换
序号|快捷键|说明
:--:|:--|:--
1|ctrl+f|当前文件查找
2|ctrl+shift+f|指定文件夹下查找
3|ctrl+r|当前文件查找并替换
4|ctrl+shift+r|指定文件夹下查找并替换
5|双击shift|全局查找指定的文件

### 暂未分类
序号|快捷键|说明
:--:|:--|:--
1|ctrl+shift+enter(回车)|语句完成(按语言自动补全)
2|ctrl+e|坚持使用的文件
3|ctrl+shift+e|最近更改的文件

## java
序号|快捷键|说明
:--:|:--|:--
1|ctrl+alt+t|快捷流程列表(if/for/try等)
2|ctrl+o|快速重写父类
3|ctrl+alt+l|选中内容快速排序
4|ctrl+d|复制选中内容到下一行
